# PhonePe Pulse Data Analysis & Visualization (Power BI)

## üìä Project Overview

This repository contains a comprehensive Business Intelligence solution built using **Microsoft Power BI** to analyze and visualize the official **PhonePe Pulse** dataset. The project focuses on transforming raw transactional and user data into an interactive dashboard, providing deep insights into market growth, user adoption, and geographical performance across India.

The primary goal of this dashboard is to make complex, large-scale data easily digestible for strategic decision-making and performance monitoring.

## ‚ú® Key Features & Insights

* **Interactive Dashboard:** A dynamic, multi-page report allowing users to filter data by year, quarter, state, and transaction category.
* **Geographical Analysis:** Utilizes map visualizations to show transactional volume and user count distribution across states and districts, highlighting high-growth areas.
* **Performance Metrics:** Tracks and compares key performance indicators (KPIs) such as Total Transaction Volume, Total Transaction Value, and Quarterly User Growth.
* **Time-Series Analysis:** Includes visualizations to track trends over time, enabling the identification of seasonal patterns and growth rates.
* **Data Modeling:** Implementation of a robust data model (e.g., Star Schema) to ensure efficient data retrieval and calculation performance.
